# Security policy

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| &lt; 1  | :x:                |

## Reporting a vulnerability

Please **do not** open a public issue for security reports.

- **Preferred:** [GitHub Security Advisories](https://github.com/adriangrahldev/express-zod-routes/security/advisories/new) (private report).
- We aim to acknowledge within a few business days and coordinate a fix and release timeline.

## Scope

Reports should concern **this package** (`express-zod-routes`). For issues in **Express**, **Zod**, or **Node.js**, use their respective security channels.
